# DishyaNirya – AR Campus Navigation Interface

## 📌 Overview
DishyaNirya is a web-based interface designed to simplify navigation within a college campus. It provides a concept for an Augmented Reality (AR) navigation system that helps users locate departments, classrooms, laboratories, and other campus facilities.

> **Note:** This project focuses on the frontend interface and UI design. It does not implement real-time AR navigation.

## ✨ Features
- Clean and responsive user interface
- Campus navigation concept
- Search for campus locations
- Interactive navigation screens
- Beginner-friendly design

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
